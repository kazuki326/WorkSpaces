# bier.jp マイページ UI設計書

## 概要

本ドキュメントでは、bier.jpマイページのユーザーインターフェース設計について定義します。直感的な操作性と視覚的な一貫性を重視したデザインを目指します。

### レイアウト仕様

| 要素 | 幅 | 高さ | 備考 |
|------|-----|------|------|
| サイドバー | 250px | 100vh | 固定配置、スクロール追従 |
| メインコンテンツ | calc(100% - 250px) | auto | 残りの幅を使用 |
| ヘッダー | 100% | 60px | 固定配置 |

## アクセシビリティ

### キーボードナビゲーション

| キー | 動作 |
|------|------|
| Tab | 次の要素へフォーカス移動 |
| Shift + Tab | 前の要素へフォーカス移動 |
| Enter/Space | ボタン・リンクの実行 |
| Escape | モーダル・ドロップダウンを閉じる |
| 矢印キー | リスト内の移動 |

### ARIA属性

```html
<!-- ナビゲーション -->
<nav aria-label="マイページナビゲーション">

<!-- 現在地表示 -->
<li aria-current="page">ダッシュボード</li>

<!-- ローディング状態 -->
<div aria-busy="true" aria-live="polite">読み込み中...</div>

<!-- モーダル -->
<div role="dialog" aria-modal="true" aria-labelledby="modal-title">
```
