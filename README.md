# bier.jp Project

このリポジトリは、bier.jpに関連する様々なプロジェクトのワークスペースを管理します。

## WorkSpaces

| WorkSpace | 概要 |
| :--- | :--- |
| [WorkSpace1_tastingnotes](./WorkSpace1_tastingnotes/) | 購入者がテイスティングノートを記録し、その平均を商品の口コミとして掲載する機能 |
| [WorkSpace2_official-line](./WorkSpace2_official-line/) | 公式LINEアカウントの構築 |
| [WorkSpace3_line-ordering](./WorkSpace3_line-ordering/) | LINE上で商品を注文できる機能 |
| [WorkSpace4_product-detail-page](./WorkSpace4_product-detail-page/) | 商品詳細ページの整理 |
| [WorkSpace5_feature-page](./WorkSpace5_feature-page/) | 特集ページの作成 |
| [WorkSpace6_beer-comparison](./WorkSpace6_beer-comparison/) | ビールの比較機能 |
| [WorkSpace7_video-content](./WorkSpace7_video-content/) | 動画コンテンツの企画・整理 |
| [WorkSpace8_search-feature](./WorkSpace8_search-feature/) | 検索機能 |
| [WorkSpace9_image-search](./WorkSpace9_image-search/) | 画像検索機能 |
| [WorkSpace10_my-page](./WorkSpace10_my-page/) | マイページ機能 |
| [WorkSpace11_recommended-products](./WorkSpace11_recommended-products/) | おすすめ商品機能 |
| [WorkSpace12_taste-graph](./WorkSpace12_taste-graph/) | 味のグラフ表示機能 |
