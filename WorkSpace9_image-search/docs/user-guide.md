# ユーザーガイド - ビール画像検索

## はじめに

ビール画像検索は、お手持ちのビールの写真から、データベース内の一致するビールを見つけることができる機能です。ラベルやボトルの写真をアップロードするだけで、ビールの詳細情報を簡単に検索できます。

---

## 使い方

### 基本的な使用手順

1. **画像を準備する**
   検索したいビールのラベルまたはボトルの写真を用意します。

2. **画像をアップロードする**
   「ビールの画像をアップロードしてください」のボタンをクリックし、画像ファイルを選択します。

3. **検索を実行する**
   「画像を検索」ボタンをクリックして、検索を開始します。

4. **結果を確認する**
   類似度の高いビールが順番に表示されます。

### 検索結果の見方

検索結果には以下の情報が表示されます：

- **ビール名**: 検索にマッチしたビールの名前
- **醸造所**: ビールを製造している醸造所名
- **類似度スコア**: アップロード画像との一致度（パーセント表示）
- **ラベル画像**: データベースに登録されているラベル画像

## 対応画像形式

### サポートされる形式

| 形式 | 拡張子 | 備考 |
|------|--------|------|
| JPEG | .jpg, .jpeg | 最も推奨される形式 |
| PNG | .png | 透過画像も対応 |
| WebP | .webp | 高圧縮率で軽量 |
| GIF | .gif | 静止画のみ対応 |
| BMP | .bmp | 対応（非圧縮） |

### 画像の推奨サイズ

| 項目 | 推奨値 |
|------|--------|
| 最小解像度 | 200 x 200 ピクセル |
| 推奨解像度 | 800 x 800 ピクセル以上 |
| 最大ファイルサイズ | 10MB |
| アスペクト比 | 制限なし（自動調整） |

### 良い画像の撮り方

精度の高い検索結果を得るために、以下のポイントを意識してください：

1. **ラベルを正面から撮影する**
   斜めからの撮影は認識精度が下がります。

2. **十分な明るさを確保する**
   暗い場所での撮影は避けてください。

3. **ピントを合わせる**
   ぼやけた画像は認識が困難です。

4. **ラベル全体を収める**
   ラベルが途切れていると認識精度が低下します。

5. **反射を避ける**
   ガラス瓶の場合、光の反射が入らないよう注意してください。

## 検索精度について

### 精度の目安

| 類似度スコア | 信頼度 | 説明 |
|--------------|--------|------|
| 90%以上 | 非常に高い | ほぼ確実に同じビール |
| 70%〜89% | 高い | 同じビールである可能性が高い |
| 50%〜69% | 中程度 | 類似したラベルデザインの可能性 |
| 50%未満 | 低い | 参考程度にご確認ください |

### 精度に影響する要因

#### 認識精度が高くなる条件

- 高解像度の画像
- 正面からの撮影
- 明るく均一な照明
- データベースに同じビールが登録されている

#### 認識精度が低くなる条件

- 画像がぼやけている
- 極端に暗いまたは明るい
- ラベルが傷んでいる、または汚れている
- 画像に複数のビールが写っている
- 限定ラベルなど特殊なデザイン

### 検索がうまくいかない場合

1. **別の角度から撮影し直す**
   ラベルが見やすい角度で再撮影してください。

2. **画像を明るくする**
   撮影環境を明るくするか、画像編集で明るさを調整してください。

3. **ラベル部分を拡大する**
   トリミングしてラベル部分だけにすると精度が向上することがあります。

## よくある質問（FAQ）

### Q: 海外のビールも検索できますか？

A: はい、データベースに登録されていれば検索可能です。現在、日本国内で流通している主要なビールと、一部の輸入ビールに対応しています。

### Q: クラフトビールは対応していますか？

A: 主要なクラフトビールには対応していますが、地域限定や小規模醸造所のビールは登録されていない場合があります。

### Q: 検索履歴は保存されますか？

A: 現在のプロトタイプ版では、検索履歴は保存されません。アップロードされた画像も、検索処理後に自動的に削除されます。

### Q: スマートフォンから利用できますか？

A: はい、スマートフォンのブラウザからもご利用いただけます。カメラで撮影した画像をそのままアップロードできます。

### Q: 検索結果が表示されない場合は？

A: 以下をご確認ください：
- インターネット接続が安定しているか
- 対応している画像形式か
- ファイルサイズが10MB以下か
- 画像が破損していないか
