# WorkSpace13: 商品ページ内の画像複数表示

## 📋 概要

商品の魅力を視覚的に伝えるための画像表示システム。メイン画像の切り替え、サムネイルによるナビゲーション、拡大表示（モーダル）、およびモバイル端末での直感的な操作（スワイプ・ドラッグ）を提供します。

## 🎯 目的・背景

### 解決する課題
- 商品画像が１枚しかないため、イメージがわきにくい
- 商品の詳細（色、質感、ラベル情報など）を拡大して確認できない
- モバイルユーザーにとって画像の切替操作が煩わしい

### 期待される効果
- **購買意欲の向上**: 複数パターンの画像確認により、商品の魅力をより深く伝える
- **UXの最適化**: スワイプやドラッグなどの直感的な操作によるストレスのない閲覧体験
- **情報の明文化**: 画像ごとの説明文により、成分表示や利用シーンなどの理解を助ける

## 📝 主要機能

### 1. メイン画像ビューアー
- **ナビゲーション**: 左右の矢印ボタンによる画像切り替え
- **スワイプ/ドラッグ**: モバイルでのスワイプ、PCでのドラッグ操作による画像遷移
- **キャプション表示**: 各画像に対応した説明文（imageDescription）をリアルタイムに更新

### 2. サムネイル・ナビゲーション
- **同期表示**: 現在表示中の画像を青枠（active）で強調
- **自動スクロール**: 選択されたサムネイルが常に表示範囲内に収まるようscrollIntoViewで制御

### 3. モーダル拡大表示機能
- **全画面表示**: 画像クリックで背景を暗転させた拡大モードへ移行
- **ページネーション**: 画面下部のドット（〇●〇）により、全体の枚数と現在位置を可視化
- **マルチデバイス対応**: タッチイベント（touchstart/touchend）とマウスイベントの両方でスワイプ操作が可能
