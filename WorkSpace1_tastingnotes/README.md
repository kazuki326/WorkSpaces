# WorkSpace1: テイスティングノート機能

## 📋 概要

購入者がビールのテイスティング（試飲）結果を記録し、それを個人の記録として蓄積できる機能。記録された評価の平均値を商品ページに「口コミ」として掲載することで、他のユーザーの購入を支援します。

## 🎯 目的・背景

### 解決する課題
- ビール購入後の体験を記録・共有する仕組みがない
- 商品ページに実際の購入者の評価が不足している
- ユーザーが自分の好みを把握しづらい

### 期待される効果
- **購入者体験の向上**: 飲酒体験を記録・整理する楽しみを提供
- **信頼性の向上**: 実際の購入者による数値評価で情報の信頼性を強化
- **販売促進**: 平均評価により「味のミスマッチ」を減らし購買を後押し
- **エンゲージメント強化**: サイトをビール体験のコミュニティに進化させる

## 📝 主要機能

### 1. テイスティングノート作成・保存
- **入力項目**: 味、香り、苦味、コク、キレ、お気に入り度（5段階評価）
- **自由記述コメント**: 個人的なメモ（非公開）
- **写真アップロード**: 複数枚対応

### 2. Myテイスティングノート（閲覧・編集）
- マイページ内に「Myビール図鑑」セクション
- 時系列・ビール名でソート/フィルタリング
- コレクション形式での表示

### 3. 平均評価の算出と表示
- 各評価項目ごとに平均値を計算
- レーダーチャートで可視化
- レビュー数も併記

### 4. バーコードスキャン（検討中）
- スマホカメラでバーコード読み取り
- 商品自動特定
- ※MVP（最小実行可能製品）では見送る可能性あり
