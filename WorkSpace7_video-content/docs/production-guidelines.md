# 制作ガイドライン

bier.jpの動画コンテンツ制作における基準とブランドガイドラインを定義します。

## 撮影基準

### 機材要件

#### 必須機材

| カテゴリ | 推奨機材 | 備考 |
|---------|---------|------|
| カメラ | 4K対応デジタルカメラ / スマートフォン（iPhone 13以降推奨） | 商品撮影は一眼レフ推奨 |
| 三脚 | 安定性の高い三脚 | 商品回転動画には回転台も使用 |
| 照明 | ソフトボックス / リングライト | 自然光も活用可 |
| マイク | ピンマイク / ショットガンマイク | ナレーション収録時は必須 |
| 背景 | 白・グレー・木目調の背景紙または撮影ボード | 商品に応じて使い分け |

#### 推奨環境

- **撮影場所:** 静かで照明をコントロールできる室内
- **撮影時間帯:** 自然光を使う場合は午前10時〜午後3時
- **温度管理:** ビールの撮影時は適切な温度を維持（泡立ちに影響）

### 映像品質基準

| 項目 | 基準 |
|------|------|
| 解像度 | 最低1080p（推奨4K） |
| フレームレート | 30fps（スローモーション部分は60fps以上） |
| アスペクト比 | 16:9（YouTube・サイト用）/ 9:16（SNSストーリーズ用） |
| ホワイトバランス | 商品の色が正確に再現されるよう調整 |
| 手ブレ | 三脚使用または手ブレ補正必須 |

### 撮影テクニック

#### 商品撮影のポイント

1. **ビール瓶・缶**
   - ラベルが正面から見えるアングル
   - 光の反射でラベルが読めなくならないよう注意
   - 複数商品は高さや色のバランスを考慮して配置

2. **グラス撮影**
   - 光の反射を活かした透明感の表現
   - ゆっくりと回転させて全体を見せる
   - ビールを注ぐシーンは泡立ちがポイント

3. **調理動画**
   - 俯瞰（真上）と斜め45度のアングルを併用
   - 手元が見やすいフレーミング
   - 完成品は美味しそうに見える照明と盛り付け

#### 人物撮影のポイント

- 目線はカメラのやや上（自然な印象）
- 背景は整理されシンプルに
- 表情は明るく親しみやすく

## 編集基準

### 編集ソフトウェア

| 用途 | 推奨ソフト |
|------|-----------|
| 本格編集 | Adobe Premiere Pro / DaVinci Resolve |
| 簡易編集 | CapCut / iMovie |
| アニメーション | Adobe After Effects / Canva |
| サムネイル | Adobe Photoshop / Canva |

### 動画構成テンプレート

#### 商品紹介動画（7-8分）

```
0:00 - 0:15  オープニング（ロゴ・タイトル）
0:15 - 0:45  今回紹介する商品の概要
0:45 - 3:00  商品の詳細（外観、ラベル、特徴）
3:00 - 5:00  実際に開封・試飲
5:00 - 6:30  味わいのレビュー・感想
6:30 - 7:30  おすすめのペアリング・シーン
7:30 - 8:00  エンディング（購入リンク・チャンネル登録促進）
```

#### 短尺商品紹介（1-2分）

```
0:00 - 0:05  タイトル表示
0:05 - 0:30  商品外観・回転
0:30 - 1:00  実際の使用シーン
1:00 - 1:30  商品の魅力ポイント（テロップ）
1:30 - 2:00  エンディング
```

#### 初心者ガイド（3-5分）

```
0:00 - 0:10  オープニング
0:10 - 0:30  今回のテーマ紹介
0:30 - 3:30  ステップバイステップの解説
3:30 - 4:30  よくある質問・注意点
4:30 - 5:00  まとめ・エンディング
```

### 編集スタイル

#### カット・トランジション

- **推奨:** シンプルなカット、クロスフェード
- **避ける:** 派手なトランジション、過度なエフェクト
- **カットのタイミング:** 無駄な間をカット、テンポよく

#### テロップ・テキスト

| 要素 | 仕様 |
|------|------|
| フォント | Noto Sans JP / ヒラギノ角ゴ |
| タイトル文字サイズ | 48-72pt |
| 説明テキストサイズ | 24-36pt |
| 文字色 | 白（黒縁取り）または bier.jpブランドカラー |
| 表示時間 | 読める速度（1文字0.1秒が目安） |

#### BGM・効果音

- **BGM:** 著作権フリー音源を使用（Artlist、Epidemic Soundなど）
- **音量:** BGMは-20dB程度、ナレーションは-6dB程度
- **効果音:** 控えめに使用、商品の魅力を引き立てる音のみ

## ブランドガイドライン

### カラーパレット

| 用途 | カラー | カラーコード |
|------|--------|-------------|
| プライマリ | ゴールド/アンバー | #D4A84B |
| セカンダリ | ダークブラウン | #4A3728 |
| アクセント | クリーム | #F5F0E6 |
| テキスト | ダークグレー | #333333 |
| 背景 | ホワイト | #FFFFFF |

### ロゴ使用規定

1. **オープニング:** 動画開始時にbier.jpロゴを表示（2-3秒）
2. **ウォーターマーク:** 動画右下に透過ロゴを常時表示（任意）
3. **エンディング:** 終了前にロゴとURL、SNSアカウントを表示
4. **余白:** ロゴ周囲に十分な余白を確保

### トーン＆マナー

#### 言葉遣い

- **親しみやすさ:** 堅すぎず、カジュアルすぎない丁寧語
- **専門用語:** 使う場合は必ず補足説明を追加
- **呼びかけ:** 「皆さん」「ビール好きの方」など

#### 表現の例

| NG | OK |
|----|-----|
| 「絶対買うべき」 | 「おすすめしたい一本です」 |
| 「まずい」 | 「好みが分かれる味わいかもしれません」 |
| 「安い」 | 「お求めやすい価格帯です」 |

### サムネイル基準

1. **構成要素**
   - 商品画像またはシーンのスクリーンショット
   - 大きく読みやすいタイトルテキスト
   - bier.jpロゴ（小さめ）

2. **テキスト量**
   - 最大10文字程度に抑える
   - モバイルでも読めるサイズ

3. **色使い**
   - ブランドカラーを基調に
   - 視認性を重視したコントラスト

## 品質チェックリスト

### 撮影完了時

- [ ] 必要なカットがすべて撮影できたか
- [ ] 音声は明瞭に録れているか
- [ ] 商品の色が正確に再現されているか
- [ ] 手ブレやピンボケがないか

### 編集完了時

- [ ] 誤字脱字がないか
- [ ] 音量バランスは適切か
- [ ] ブランドカラー・フォントが統一されているか
- [ ] オープニング・エンディングが入っているか
- [ ] 動画の長さは適切か

### 公開前

- [ ] タイトル・説明文は最適化されているか
- [ ] サムネイルは魅力的か
- [ ] タグ・カテゴリは適切に設定されているか
- [ ] 関連リンクは正しいか
- [ ] 複数デバイスでの表示確認をしたか

## ファイル命名規則

```
[カテゴリ]_[コンテンツ名]_[バージョン]_[日付].[拡張子]

例:
product_weihenstephaner-hefeweiss_v1_20250115.mp4
guide_beer-pouring_v2_20250120.mp4
culture_munich-brewery_final_20250125.mp4
```

## 制作フロー

1. **企画（1週間前）**
   - コンテンツ内容の決定
   - 台本・構成案の作成
   - 必要機材・商品の準備

2. **撮影（3-5日前）**
   - セッティング・リハーサル
   - 本番撮影
   - 素材の確認・バックアップ

3. **編集（2-3日）**
   - カット編集
   - テロップ・BGM追加
   - 色調整・仕上げ

4. **レビュー（1日）**
   - チェックリストに基づく確認
   - 修正対応

5. **公開**
   - 各プラットフォームへのアップロード
   - メタデータの設定
   - SNSでの告知
